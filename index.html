伝言ゲームの仕組みを取り入れたプログラミング学習アプリとは

このアプリは、お絵かき伝言ゲーム「Gartic Phone」を参考にした、伝言ゲーム形式のプログラミング学習アプリです。
「文章 → コード(実行結果) → 文章 → コード(実行結果) → 文章」と、文章とコードを交互に伝えていきます。プログラミングを担当する人は、p5.jsを使ってお題に沿った絵をコードで描きます。ゲーム感覚で楽しく進めながら、自然とプログラミングスキルを身につけられるのが特徴です。

Point

ゲーム感覚で楽しく学べる

お題に沿ってコードを書いたり、ほかの人のコードの結果を見て想像したりと、遊び感覚で進めることができます。ゲームのようにワクワクしながら、プログラミングに親しめます。

実践的なスキルが身につく

p5.jsで実際にコードを書くので、手を動かしながら学べます。コードを書く力やエラー解決のスキルが向上し、実践的なスキルが自然と身につきます。

仲間と一緒に楽しく学べる

お互いのアイデアが反映されることで、時には意外な作品が生まれることもあり、予想外の展開に笑いが広がります。仲間と一緒に進めるからこそ、遊びながら楽しく学べます。

Gartic Phoneとは

Gartic Phoneは、全員同時にお題を「文章⇒絵⇒文章⇒絵⇒…」と交互に伝えていき、最後にお題がどのように変わっていったのかを見て、感想戦で盛り上がれるゲームです。

4人目のコード(日の出)

        let sunY;
        function setup() {
          createCanvas(300, 300);
          sunY = height + 25;
        }
        function draw() {
          for (let i = 0; i < height; i++) {
            let r = map(i, 0, height, 100, 255); 
            let g = map(i, 0, height, 150, 150);
            let b = map(i, 0, height, 200, 100);
            stroke(r, g, b);
            line(0, i, width, i);
          }
          fill(255, 200, 0); 
          noStroke();
          ellipse(width / 2, sunY, 50, 50);
          sunY -= 0.5; 
          if (sunY <= 100) {
            sunY = 100;
          }
          fill(0);
          ellipse(150,350,500,150);
        }
